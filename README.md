## 练习 1：学员花名册管理器 `roster.py`

用列表和字典存储学员信息的命令行管理工具。

**功能**

- ✅ 添加学员（姓名、邮箱、加入日期）
- ✅ 查看所有学员
- ✅ 按姓名删除学员
- ✅ 无效输入时提示，不崩溃

**局限**

- 数据未持久化，程序退出后数据丢失
- 暂不支持查询单个学员

**运行方式**

```
python roster.py
```

## 练习 2：文本词频统计器 `word_count.py`

输入一段文字，统计每个单词出现次数，按频率从高到低排序输出。

**功能**

- ✅ 忽略大小写差异
- ✅ 按出现次数从高到低排序
- ✅ 输入为空时友好提示

**运行方式**
python word_count.py

## 练习 3：待办事项清单 `003_todo_list.py`

带本地文件保存的命令行待办工具，程序重启后数据不丢失。

**功能**

- ✅ 添加待办事项
- ✅ 标记完成并移除
- ✅ 查看当前清单
- ✅ 数据保存到 `todos.json`，重启后自动加载
- ✅ 文件不存在时不报错

**运行方式**
python 003_todo_list.py

## 练习 4：安全的计算器 `004_Calculator.py`

支持四则运算，任何错误输入都不会导致程序崩溃。

**功能**

- ✅ 支持 + - * / 四则运算
- ✅ 除数为零时提示，不崩溃
- ✅ 输入非数字时提示，不崩溃
- ✅ 输入 quit 优雅退出

**运行方式**
python 004_Calculator.py
